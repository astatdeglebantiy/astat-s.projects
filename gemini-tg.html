Gemini bot in telegram

Multifunctional Gemini API integration in telegram.

×

There are settings: language, reset chat, set Gemini model, set first prompt, set null message and set Gemini API key.

Language

Setting the language, to set it you need to press the button and write the language from the suggested ones within 30 seconds.

Reset chat

Clear chat, if you click it, it will be like hitting the bot on the head with a sledgehammer and it will forget everything you talked about.

Set Gemini model

Setting the Gemini model, you can see the available ones at this link, for example gemini-2.0-flash.

Set first prompt

Sets a system prompt (like an instruction for a bot), and works only after a reset.

Set Gemini API key

Setting the Gemini API key, this key can be generated from this link.
